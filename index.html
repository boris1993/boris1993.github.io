JVM运行时的数据区域

发表于 2019-03-27 更新于 2019-04-01 分类于 学习记录 ， Java ， 基础知识

JVM在运行时，会按照程序执行的需要来创建一系列的运行时数据区域。有的区域只会随JVM起停而被创建和销毁，有的区域则会独立分配给各个线程，并随线程的起停而创建和销毁。这些运行时区域，按照功能和性质不同，会分成如下几部分：

  • 线程专有
    • pc(program counter)寄存器
    • JVM栈
    • 本地方法栈
  • 线程间共享
    • 堆
    • 方法区
      • 运行时常量池
