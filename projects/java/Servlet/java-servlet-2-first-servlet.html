发表于 2019-03-06 更新于 2019-03-22 分类于 学习记录 ， Java ， Servlet

Servlet（Server Applet），全称Java Servlet，是用Java编写的服务器端程序。其主要功能在于交互式地浏览和修改数据，生成动态Web内容。本系列将一步步地写出一个Servlet程序。

这篇博文将演示如何创建一个servlet。
