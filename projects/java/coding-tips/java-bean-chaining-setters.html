用 Lombok 实现 setter 的链式调用

Lombok 是一个自动生成模版代码的工具，像构造方法啊、getter 和 setter 啊，都可以用一个或多个注解来代替，然后在编译期由 Lombok 替你写这部分的代码。如果你还没用过，那么请务必学习和试用一下，它可以为你省下那些浪费在模版代码上的时间。
