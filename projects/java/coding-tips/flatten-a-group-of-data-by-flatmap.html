使用FlatMap“展平”一系列数据

发表于 2019-09-26 更新于 2019-12-17 分类于 学习记录 ， Java ， 代码技巧 阅读次数： Valine：

在处理业务数据时，偶尔会需要从一系列逗号分隔的数据中取出各个元素并去重。本文将介绍如何通过Java 8中的FlatMap简化这个操作。

介绍需求

假设现在有这样子的一个排班信息：

日期 班组
2019-01-02 Unit-2, Unit-3
2019-01-03 Unit-2
2019-01-04 Unit-1, Unit-3
2019-01-01 Unit-2, Unit-3
2019-01-05 Unit-3
2019-01-06 Unit-1, Unit-2
2019-01-07 Unit-1
2019-01-08 Unit-2, Unit-3

而目标是，从中取出这个表中的所有班组，并去重和排序，即最终的结果是Unit-1, Unit-2, Unit-3。
